## Apresentação do aplicativo

<p>
  Este app foi desenvolvido com base no vídeo do YouTube <a href="https://www.youtube.com/watch?v=iV0ovqpFB7U">Consumindo api do via cep com retrofit.</a>
  A finalidade do aplicativo é o treinamento de consumo de API, utilizando o Retrofit. 
  Os dados não estão sendo persistidos, então, ao reiniciar o app, o conteúdo não permanece salvo. 
  A implementação da base de dados será realizada em breve, contudo, o projeto está com o funcionamento esperado e pode ser utilizado.
</p>

### Tecnologias - recursos utilizados nesse aplicativo:
<ul>
  <li><b>Kotlin</b>;</li>
  <li><b>Navigation Component:</b> Uma activity contém vários fragments em vez de criar várias activities;</li>
  <li><b>View Binding:</b> Em vez de inflar as visualizações manualmente, a vinculação de visualizações cuidará disso;</li>
  <li><b>Hilt:</b> Para injeção de dependência.</li> 
</ul>
